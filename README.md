# 📧 Email Spam Classification System

## 📌 Project Description
This project implements a machine learning-based email classification system to detect spam and non-spam emails. The system is developed using Python and applies two classification algorithms: K-Nearest Neighbors (KNN) and Naïve Bayes.

The objective of this project is to analyze the effectiveness of these algorithms in identifying spam emails based on extracted textual features.

---

## 📊 Dataset
The dataset used in this project is the **Spambase Dataset** from the UCI Machine Learning Repository.

🔗 Dataset Link:  
https://archive.ics.uci.edu/ml/machine-learning-databases/spambase/spambase.data

### Dataset Details:
- Number of Instances: 4601
- Number of Features: 57
- Target Variable: 
  - 1 = Spam
  - 0 = Non-Spam

---

## ⚙️ Algorithms Used

### 1. K-Nearest Neighbors (KNN)
- Classifies data based on the nearest neighbors
- Sensitive to feature scaling
- Uses Euclidean distance

### 2. Naïve Bayes
- Based on probability theory
- Assumes feature independence
- Fast and efficient
